# Seekho-Sikhhao

Seekho-Sikhhao is a community-driven peer learning platform designed to enable students to teach, learn, and exchange practical skills through short interactive sessions. The platform combines skill sharing, networking, and community building to transform campus knowledge into a collaborative learning ecosystem.

Developed for the IIT Jodhpur community, Seekho-Sikhhao aims to make learning more accessible, affordable, and collaborative while empowering students to become both learners and educators.

## Problem Statement

Students often possess valuable practical skills and knowledge that remain underutilized due to the absence of a structured platform for informal learning and skill exchange. Existing learning platforms are frequently expensive, impersonal, or focused primarily on academic content.

Additionally, opportunities for interdisciplinary interaction, peer mentorship, and community-driven learning are often limited within campus environments.

## Solution

Seekho-Sikhhao provides a structured ecosystem for peer-to-peer learning through short, skill-focused sessions. Students can create classes, discover learning opportunities, book sessions, request specific skills, and participate in a collaborative learning community.

## Key Features

* Skill-sharing session creation and management
* Student-led micro-learning classes
* Online and offline learning support
* Class discovery and booking system
* Skill request and matchmaking board
* Wallet-based micro-payment system
* Instructor earnings tracking
* QR-based attendance verification
* Campus room booking integration
* AI-assisted recommendations and moderation support
* Google Authentication with IIT account access
